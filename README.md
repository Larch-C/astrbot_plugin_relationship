## 📦 安装

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_relationship

# 控制台重启AstrBot
```

## ⌨️ 使用命令

### 基础命令
```plaintext
/好友列表  
/群列表
```
## 📌 注意事项
1. 建议在artrbot配置->消息平台->转发消息的字数阈值(forward_threshold) 设置低一些，防止本插件刷屏，推荐设为150
